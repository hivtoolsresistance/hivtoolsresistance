### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hivtoolsresistance/hivtoolsresistance.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run 
  ```sh
  ng serve
  ```
